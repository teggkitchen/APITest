# Android 會員購物演示



## 運行專案前置

下載專案

把專案build起來

## 版本

Gradle版本：**3.1.2**

minSdkVersion：**21**

targetSdkVersion：**28**

## 備註

這邊要更改IP
</br>
否則連不到API主機

檔案路徑：
```cmd
APITest/app/src/main/java/com/teggkitchen/apitest/config/Config.java 
```

更改這一行：
```cmd
public final static String URL_DEBUG = "http://192.168.43.42:8000";
```
